# 財金所順利畢業 SOP (NTUST Finance Graduate Institute Graduation Guide)

這是一個提供給國立臺灣科技大學（臺科大）財務金融研究所學生的畢業流程指南網頁專案。主要目的是協助研究生清楚掌握從入學到畢業各個階段的任務、必備文件及注意事項，減輕準備畢業手續時的焦慮與負擔。

本專案目前發布於 GitHub Pages。

## 🚀 專案功能與特色

- **完整流程指南**：詳細列出「入學後一年內」、「口試前」、「口試當天」及「口試後」四個核心階段的標準作業流程 (SOP)。
- **表單下載區**：彙整所有畢業相關的申請表單，方便學生直接點擊下載使用。
- **實用工具清單**：提供如「流程檢核表」等實用資源，幫助學生追蹤自身進度。
- **響應式設計 (RWD)**：支援桌上型電腦、平板及智慧型手機等各種裝置瀏覽，排版會自動適應螢幕大小。
- **無障礙輔助功能**：內建字體大小調整功能（A- / A+ / 重設），提升閱讀體驗。
- **圖片燈箱效果**：點擊網頁中的圖片或截圖即可放大檢視，方便閱讀系統操作步驟或表單填寫範例。

## 📂 網站架構

專案主要由以下幾個靜態頁面組成：

- `GraduationGuide.html`：網站首頁，提供完整的畢業流程四階段說明。
- `FormDownloads.html`：表單下載區，包含各階段所需的 PDF 或 Word 檔案連結。
- `UpdateLog.html`：更新日誌，記錄網站與系所規定的修訂紀錄。

### 目錄結構說明

- `forms/`：存放所有提供給學生下載的表單檔案（如：論文指導同意書、口試申請表等），依階段分類。
- `images/`：存放網頁所需的圖片素材、各類系統操作截圖、表單範例以及 favicon 圖示。

## 🛠 使用技術

本專案採用純靜態網頁技術開發，輕量且易於維護：

- **HTML5**：負責網頁的語意化結構。
- **CSS3**：負責網頁的樣式設計與響應式排版（樣式直接寫在各 HTML 檔案內的 `<style>` 標籤中）。
- **JavaScript (Vanilla JS)**：負責網頁的互動功能，包含圖片燈箱 (Lightbox)、字體大小調整、返回頂部按鈕，以及清除 URL 錨點等功能（腳本同樣包含在 HTML 中）。

## 💻 如何運行與開發本專案

由於這是一個純靜態前端專案，無需安裝任何後端伺服器環境或依賴套件。

1. 將本專案複製 (Clone) 或下載解壓縮至本機電腦。
2. 直接點擊兩下開啟 `GraduationGuide.html` 檔案，即可在預設瀏覽器中瀏覽。
3. **建議的開發方式**：使用 Visual Studio Code 編輯器，並安裝「Live Server」擴充功能。在 HTML 檔案上點擊右鍵選擇 `Open with Live Server`，如此一來存檔時瀏覽器便會自動重新整理，提供更好的開發體驗。

## 📝 維護與更新說明

若未來系所規定有所更改，建議網站維護者依循以下步驟進行更新：

1. **修改內容**：使用編輯器開啟對應的 HTML 檔案，尋找該階段的文字進行修改。
2. **更新表單檔案**：若有新增或修改相關表單，請將新的檔案放入 `forms/` 資料夾對應的分類中，並確認 `FormDownloads.html` 及各頁面的檔案連結更新為正確路徑。
3. **新增圖片**：若需更新系統截圖，請將新圖片放入 `images/` 資料夾，並以 `class="thumbnail"` 套用樣式以支援燈箱效果。
4. **填寫更新日誌**：每次修改後，請務必於 `UpdateLog.html` 中新增一筆紀錄，註明修改日期與變更細節，以利後續學弟妹確認資訊的時效性。
